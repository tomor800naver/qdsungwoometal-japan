# 웹사이트 내용 편집 가이드

이 프로젝트는 **React + Vite**로 구성되어 있으며, 주요 텍스트와 이미지는 각 페이지 파일에서 직접 수정할 수 있습니다.

## 1. 주요 편집 파일 위치

모든 페이지 파일은 `client/src/pages/` 디렉토리에 위치합니다.

| 페이지 | 파일 경로 | 주요 내용 |
| :--- | :--- | :--- |
| **메인 홈** | `client/src/pages/Home.tsx` | 히어로 섹션 문구, 회사 소개, 주요 장점 |
| **제품 소개** | `client/src/pages/Products.tsx` | 제품 목록, 상세 설명, 이미지 |
| **제조 공정** | `client/src/pages/Manufacturing.tsx` | 설비 정보, 품질 관리 내용 |
| **문의하기** | `client/src/pages/Contact.tsx` | 주소, 연락처, 지도 정보 |

## 2. 편집 방법

### 텍스트 수정
각 `.tsx` 파일 내에서 수정하고 싶은 텍스트를 찾아 직접 변경하세요.
- 예: `Home.tsx`의 `metrics` 배열 내 `label` 값을 수정하여 수치를 변경할 수 있습니다.

### 이미지 수정
파일 상단에 정의된 이미지 URL 변수를 수정하세요.
- 예: `const heroImage = "..."` 부분의 URL을 새로운 이미지 주소로 교체합니다.

## 3. 변경 사항 확인
파일을 수정한 후, 제가 제공해 드린 **로컬 미리보기 링크**를 통해 실시간으로 변경된 내용을 확인하실 수 있습니다.

**편집을 원하시는 특정 문구나 이미지가 있다면 저에게 말씀해 주세요. 제가 대신 수정해 드릴 수도 있습니다!**
